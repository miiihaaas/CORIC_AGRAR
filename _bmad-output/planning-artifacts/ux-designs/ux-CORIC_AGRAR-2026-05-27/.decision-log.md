## 2026-05-27 — Batch 1 odgovori

### Vizuelni identitet & strategija

- **1a) Slogan zadržan:** *"Prijatelj koji razume zemlju!"* je hero slogan na Početnoj. Iz Projektnog zadatka, ne menja se.
- **2a) Light mode samo.** Bez dark mode-a u v1. Fokus na zelenu paletu agrarnog sektora.
- **3b) Lemken inspiracija — širi obim:** patterns za filtering nav, kategorijske kartice, vizuelna hijerarhija + tipografski pristup (veliki bold naslovi, ozbiljan ton). Visual reference za UX, ne mehanika kopiranja.

### Komponente & layout

- **4a) Sticky meni — shrunk on scroll-down:** pri skrolu nadole se kondenzuje (manji logo, kompaktnija visina), pri skrolu nagore se proširuje.
- **5a) Top header mobile — kondenzovan:** samo ikona telefona, klik proširi sve. Adresa i puni telefon vidljivi na tablet i desktop.
- **6a) "O nama" intro sekcija na Početnoj:** dodajemo pun blok između hero-a i Traktori sekcije — naslov, kratak tekst, CTA *Saznaj više* (vodi na /o-nama). **Ovo se prebacuje u EXPERIENCE.md Information Architecture i u FR-1 ekstenziju.**
- **7a) Filteri godišta i cene:** range slider (min-max) za oba.
- **8a) Motor akordion default-open:** prva sekcija (Motor) je proširena na load, ostale (Transmisija/Hidraulika/Ostalo) skupljene. Korisnik može da skupi Motor i otvori druge.
- **9a) Footer "Najnovije vesti" kolona — dinamička:** automatski prikazuje 3 najnovije blog objave (po `created_at`).

### Interakcije

- **10a) Vremenska lenta — animirana SVG / CSS-only.** Bez JS biblioteke. Animacija pri scroll-into-view (intersection observer).
- **11b) Vizuelni marker za i18n fallback — tooltip sa info ikonom (`ⓘ`):** pored teksta polja koje koristi fallback, mali ikonski indikator. Hover/click otvara tooltip "Sadržaj na srpskom jeziku — još nije preveden".
- **12b) Social share dugmad (blog post):** Facebook + Viber + WhatsApp + Copy link. Pozicija — sticky leva strana na desktop, ispod naslova na mobile (`[ASSUMPTION]`).
- **13b) Open Graph slike — admin upload po proizvodu/objavi.** Manualna kontrola; fallback na generic OG sliku (logo + slogan) ako admin nije upload-ovao.
- **14a) Brošura card layout — outlined:** cover-thumbnail levo, naslov + file-size + format desno ("2.8 MB, PDF"), CTA *PREUZMI*. Vidi mockup Stranica proizvoda 1.0.

## 2026-05-27 — Token ekstrakcija iz materijala

### HTML prototip (`.working/extract-html-tokens.md`)

- **Paleta:** dva slična tamna zelena (`#25402f` primarni, `#1f3f2f` card bg), srednji `#395f48`, svetli `#4d7e60`. Akcenti: lime `#c8d32c`, gold `#e7af12`. Jeegee plava `#00A4E9`.
- **Tipografija:** **Roboto** (Google Fonts, 300/400/700) — single family, **nema posebne display fonte**. PRD § 4.1 FR-1 i Projektni zadatak pominju "tanki i elegantni" — Roboto 300 ispunjava taj zahtev.
- **Base body:** `1.25rem` (20px), override Bootstrap default.
- **Container:** max 1200px. Bootstrap 5.3.2 breakpoints 768 / 1200.
- **Radii:** 5/6/8/10px + 999px pill. **`[ASSUMPTION]`** unifikujem u 4-step skala.
- **Gap-ovi nađeni u prototipu:** nema CSS variables (token system se gradi iz nule); nema focus styles; nema `prefers-reduced-motion`; outline-only buttons ruše focus indikator → **a11y rad treba u dev fazi**.

Mockup pokazuje **unified tamno-zeleni UI chrome** (nav, footer, dashboard). → **Rešenje:** brand boja se primenjuje samo na *Ponavljajući element* u hero pozicijama na brend/proizvod stranama (npr. Jeegee plavi hero card); ostatak UI ostaje unified tamno-zeleni. Konsistentno sa PRD §4.4 FR-9 koji to već implicira ("Hero koristi plavu varijantu...").

  4. **FR-19 polja:** već razrešeno u PRD finalize fazi — zadržavamo 5 polja (Batch 2 odluka), mockup se ažurira.

- **Sparse wave dividers:** waves se koriste samo na "Priče sa polja" top edge i ispod "Preuzmi katalog" CTA — nije sveprisutan pattern. Treba zadržati selektivno.
